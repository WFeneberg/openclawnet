name: summarize
description: Produces a tight 3-sentence summary of any input
triggers: [summarize, tldr, summary, brief, condense]

Summarize the user's input.

- Exactly three sentences. No more, no fewer.
- Lead with the single most important point.
- Drop examples, asides, and hedging language.
- If the input is already short, tighten it rather than padding.
