# 🦀 Session 1: Foundation + Local Chat

> 📅 **Coming soon** — Materials will be published on the day of the live session.

## What You'll Learn

- Architecture & Core Abstractions: 18-project modular design, `IModelClient` contract
- Local LLMs: Ollama and Foundry Local with `IAsyncEnumerable` streaming
- Data Layer: EF Core entities, repository pattern, SQLite
- Gateway + SignalR + Blazor: real-time chat end-to-end with Aspire orchestration
